Student Marks Analyzer (Python)

📌 Project Description

This project is a simple program built using Python that analyzes student marks.
It takes marks as input from the user and calculates useful statistics such as the average, highest, and lowest marks.

🚀 Features

- Accepts marks of multiple students
- Calculates average marks
- Finds the highest mark
- Finds the lowest mark
- Simple and beginner-friendly program

🛠 Technologies Used

- Python

▶️ How to Run the Program

1. Install Python on your system.
2. Download the file "student_marks_analyzer.py".
3. Open Python IDLE or a terminal.
4. Run the program using the command:

python student_marks_analyzer.py

💡 Example Output

Enter mark of student: 90

Lowest mark: 72

This project demonstrates:

- Taking user input
